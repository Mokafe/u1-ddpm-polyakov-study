# Phase 2a

## Purpose
Beta-conditioned DDPM generation of 32x32 Polyakov loop maps.

## Main files
- `notebooks/phase-2a-ddpm-for-32-32-polyakov-maps.ipynb`
- `reports/Phase2a_実験ノート_図入り整合版.docx`

## Focus
- DDPM training
- beta conditioning
- reproduction of the order parameter mean \( M = |\bar{P}| \)
- held-out beta interpolation
- unit-circle consistency check

## Notes
- This phase focuses on first-order statistics.
- Distribution, correlation, and topology checks are handled in later phases.
